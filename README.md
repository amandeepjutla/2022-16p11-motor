# "Motor difficulties in 16p11.2 copy number variation" analytic code

`./jupyter/analysis.ipynb` is a notebook that can reproduce the analyses from our paper ["Motor difficulties in 16p11.2 copy number variation"](https://doi.org/10.1002/aur.3132). It assumes that the Simons Searchlight datasets (available from SFARIbase) are located in `./data/raw`.
